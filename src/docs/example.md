## 4. PC 端功能模块（示例摘录）

### 4.3 认证中心 — 个人信息填报（字段示例）

| 字段 | 标识 | 类型 | 必填 | 限制 |
|------|------|------|------|------|
| 姓名 | real_name | string(20) | 是 | 2-20 字 |
| 身份证号 | id_card | string(18) | 是 | 18位+校验 |
| 手机号 | phone | string(11) | 是 | 手机号格式 |
| 专业方向 | speciality | enum[] | 是 | 多选，最多3项 |

### 4.4 证书管理 — 证书申领步骤

```
Step 1：选择证书类型（演员资格证 / 专业技术资格证书）
Step 2：选择认定方式 + 上传材料
Step 3：费用确认与支付
Step 4：提交申请
```

### 4.5 职业能力评价 — 考试防作弊规则

| 规则 | 说明 |
|------|------|
| 防切屏 | 切出页面记录次数，≥3 次强制交卷 |
| 断线续考 | 网络断开后 5 分钟内可恢复 |
| 自动交卷 | 倒计时结束系统自动提交 |

## 6. 核心业务流程（示例：证书申领）

```
演员发起申领 → 选择类型 → 上传材料 → 支付 → 提交
→ Admin审核 → 艺德培训 → 艺德考核 → 签发证书 → 完成
```
