# 培训管理台

## 📋 业务与功能

### 1.1 核心目标
> 管理员配置和管理培训课程的后台页面。支持课程的创建、发布、下架、学员进度查看等操作。对应需规 4.5.3 节。

### 1.2 功能清单
- **统计卡片区**：展示已发布课程数、本月活跃学员数、平均完成率 3 个关键指标
- **筛选区**：支持按课程分类、课程状态、是否必修、关键词筛选
- **数据表格**：展示课程列表（序号、课程名称、分类、是否必修、总课时、已报名人数、完成率、状态、创建时间、操作）
- **行级操作**：编辑、发布、下架、查看学员、删除
- **课程详情弹窗**：查看课程的详细信息和课程大纲
- **课程编辑弹窗**：编辑课程基本信息
- **学员进度弹窗**：查看学员学习进度列表

### 1.3 交互要点
- Tab 导航：点击导航菜单 → 跳转对应页面
- 搜索按钮：点击 → 按条件过滤表格数据
- 重置按钮：点击 → 清空所有筛选条件
- 查看详情：点击课程名称或"查看"按钮 → 打开课程详情弹窗
- 编辑：点击"编辑"按钮 → 打开课程编辑弹窗
- 发布/下架：点击按钮 → 二次确认弹窗 → 变更课程状态
- 查看学员：点击"查看学员"按钮 → 打开学员进度弹窗
- 删除：点击"删除"按钮 → 二次确认弹窗（仅报名人数=0时可用）

```
培训管理台
```

### 2.2 数据来源
- **数据类型**：课程管理列表
- **数据源**：生成示例数据（MOCK_DATA）
- **关键字段**：
  - `course_id`: 课程ID
  - `course_name`: 课程名称
  - `category`: 课程分类（ethics/performance/law/psychology）
  - `is_required`: 是否必修
  - `total_lessons`: 总课时数
  - `enrolled_count`: 已报名人数
  - `completion_rate`: 完成率
  - `status`: 状态（draft/published/offline）
  - `instructor_name`: 讲师姓名
  - `created_at`: 创建时间
