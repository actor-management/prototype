# 中国演艺人才管理与服务平台 - 认证管理台

## 📋 业务与功能

### 1.1 核心目标
> 管理员用于处理用户认证申请的后台管理页面。支持查看认证队列、单条审批/批量审批、导出数据、查看审批统计。是 Admin 日常工作的高频使用页面，需要高效的信息筛选和批量操作能力。

### 1.2 功能清单

- **统计栏**：4 个统计卡片（待审核数、今日已处理、本月通过率、平均处理时长）
- **筛选区**：多条件筛选（状态、用户类型、姓名、手机号、身份证号、日期范围）
- **数据表格**：分页表格展示认证申请列表，支持排序
- **行级操作**：查看详情、审核通过、审核驳回
- **批量操作**：全选/取消全选、批量通过、批量驳回
- **导出功能**：导出当前筛选结果为 Excel
- **审核详情弹窗**：查看申请人完整信息，填写审核意见

### 1.3 交互要点

- 筛选条件变更后实时刷新列表
- 表格行 hover 高亮
- 批量操作需二次确认弹窗
- 审核驳回须填写驳回原因
- 统计卡片展示实时数据

### 2.2 数据来源

- **数据类型**：列表数据、统计指标
- **数据源**：内置示例数据
- **关键字段**：
  - `real_name`: 申请人姓名
  - `user_type`: 用户类型
  - `phone`: 手机号（脱敏）
  - `id_card`: 身份证号（脱敏）
  - `apply_time`: 申请时间
  - `status`: 审核状态
  - `reviewer_name`: 审核人
  - `review_time`: 审核时间

## 🎨 布局与结构

### 3.1 整体布局

- **布局模式**：单栏（顶部导航 + 主内容区 + 页脚）
- **容器宽度**：最大宽度 1440px，居中展示
- **关键尺寸**：
  - 统计卡片：4 列栅格
  - 筛选区：3 列表单布局
  - 表格：全宽

### 3.2 响应式适配

- **桌面端（≥1200px）**：标准布局
- **平板端（768-1199px）**：筛选区 2 列，统计卡片 2 列
- **移动端（<768px）**：单列堆叠
