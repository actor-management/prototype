# 演艺人才平台 - 移动端首页

## 📋 业务与功能

### 1.1 核心目标
>
> 移动端首页是用户登录后的第一落地页，根据三种角色（Admin/Actor/User）展示差异化内容，聚合待办事项、通知公告、党建专栏等核心模块，并通过底部导航提供全局跳转能力。

- **Admin（管理员）**：审批看板 + 待审核待办 + 通知公告
- **Actor（演员）**：证书/培训/考核快捷入口 + 缴费/培训类待办
- **User（普通用户）**：浏览查询入口 + 无待办模块（改为消息动态概念），未认证时展示认证引导

### 1.2 功能清单

- **顶部欢迎区**：用户头像 + 姓名 + 欢迎语 + 认证状态标签 + 角色切换标签（朱红渐变背景）
- **认证引导卡片**（仅 User 未认证时）：醒目渐变卡片，含「立即认证」按钮
- **快捷操作宫格**：4 个按角色差异化的图标入口
- **待办事项**（仅 Admin/Actor）：纵向卡片列表，点击弹出详情弹窗
- **通知公告**：标题列表（最新 5 条），点击弹出通知详情全屏弹窗
- **党建专栏**：单条图文卡片，点击弹出文章详情全屏弹窗
- **底部导航**：按角色差异化的 Tab 栏（Admin 6Tab / Actor 6Tab / User 5Tab）

### 1.3 交互要点

- 点击角色标签 → 跳转到登录页 (`/prototypes/yanyuan-app-login`) 重新选择角色
- 点击快捷操作 → 跳转到对应功能页面
- 点击待办卡片 → 弹出底部半屏待办详情弹窗
- 点击待办「查看更多 >」→ 弹出全屏待办列表弹窗（含筛选标签：全部/待处理/已处理）
- 点击通知 → 弹出全屏通知详情弹窗
- 点击通知「查看更多 >」→ 弹出全屏通知列表弹窗（含筛选标签：全部/证书发放/政策公示/考试安排）
- 点击党建卡片 → 弹出全屏党建文章详情弹窗
- User 未认证时，点击底部「验证」「学习」「我的」Tab → 弹出认证引导弹窗
- 底部导航点击 → 跳转到对应路由

### 2.2 数据来源

- **数据源**：生成示例数据（原型演示用）
- **角色信息**：从 `localStorage` 的 `yanyuan_role` 读取
- **关键字段**：
  - `role`: admin/actor/user
  - `userName`: 角色对应的模拟姓名
  - `authStatus`: 认证状态（authenticated/pending/rejected/none）

## 🎨 布局与结构

### 3.1 整体布局

- **布局模式**：单栏纵向滚动（移动端 375px 设计宽度）
- **容器宽度**：100% 流式
- **关键尺寸**：
  - 顶部欢迎区：高度约 140px，朱红渐变背景
  - 快捷操作宫格：4 列等宽网格
  - 待办卡片：圆角卡片列表
  - 底部导航：固定底部 56px

### 3.2 响应式适配

- **移动端（<768px）**：唯一目标尺寸，全部模块单栏排列

### 4.2 自定义设计要点

**自定义色彩**：

- 朱红主色：`#C41A1A` → `#E53935` 渐变 — 顶部欢迎区背景
- 证书橙：`#FF8C00` — 快捷操作图标
- 认证紫：`#7B1FA2` — 认证相关标签
- 培训蓝：`#1976D2` — 培训相关图标
- 考核绿：`#2E7D32` — 考核相关图标

### 4.3 组件状态

- **待办卡片**：默认白底 → 按下微缩 → 已过期红色提示
- **底部 Tab**：默认灰色 → 激活态主色 + 放大图标
- **快捷操作图标**：默认圆形背景 → 按下缩放动画

## ⚙️ Axure API 说明

### 5.1 事件列表（eventList）

| 事件名称 | Payload 类型 | 触发时机 | 说明 |
|---------|-------------|---------|------|
| `on_tab_change` | `string` | 底部 Tab 切换 | 返回 Tab 索引 |
| `on_todo_click` | `string` | 点击待办项 | 返回待办 ID |
| `on_notice_click` | `string` | 点击通知项 | 返回通知 ID |

### 5.2 动作列表（actionList）

| 动作名称 | Params 类型 | 参数说明 | 功能描述 |
|---------|------------|---------|---------|
| `switch_role` | `string` | admin/actor/user | 切换演示角色 |

### 5.3 变量列表（varList）

| 变量名称 | 类型 | 默认值 | 说明 |
|---------|-----|-------|------|
| `current_role` | `string` | `actor` | 当前角色 |
| `auth_status` | `string` | `authenticated` | 认证状态 |
| `active_tab` | `number` | `0` | 当前激活 Tab |

### 5.4 配置项列表（configList）

| 配置项名称 | 类型 | 默认值 | 说明 |
|----------|-----|-------|------|
| `user_name` | `string` | `张明远` | 用户姓名 |
| `user_avatar` | `string` | `` | 用户头像URL |
| `auth_status` | `string` | `authenticated` | 认证状态 |

### 5.5 数据项列表（dataList）

**待办事项**

```typescript
{
  id: string;         // 待办ID
  title: string;      // 待办标题
  type: string;       // 类型：payment/training/exam/renewal/review
  deadline: string;   // 截止时间
  action_url: string; // 跳转链接
}
```

**通知公告**

```typescript
{
  id: string;           // 通知ID
  title: string;        // 通知标题
  category: string;     // 分类
  publish_time: string; // 发布时间
  is_read: boolean;     // 是否已读
}
```

**党建文章**

```typescript
{
  id: string;           // 文章ID
  title: string;        // 标题
  summary: string;      // 摘要
  cover_image: string;  // 封面图
  publish_time: string; // 发布时间
}
```
