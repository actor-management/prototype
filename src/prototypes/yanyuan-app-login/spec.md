# 演艺人才平台 - 登录/授权页

## 📋 业务与功能

### 1.1 核心目标
> 平台统一登录入口，支持微信一键手机号登录和三角色演示快捷登录，引导用户快速进入平台。

### 1.2 功能清单
- **品牌展示**：平台 Logo + 名称 + 标语
- **微信登录**：微信一键获取手机号按钮（生产环境主入口）
- **演示登录**：三角色快捷登录（管理员/演员/普通用户）
- **用户协议**：勾选框 + 用户协议/隐私政策链接

### 1.3 交互要点
- 必须勾选用户协议才可点击任何登录按钮
- 未勾选时点击登录按钮：弹出提示"请先阅读并同意用户协议"
- 登录成功后跳转首页

---

## 📊 内容规划

### 2.1 信息架构
```
登录页
├── 品牌区（Logo + 名称 + 标语）
├── 微信登录按钮
├── 分隔线（"或使用演示账号"）
├── 三角色演示按钮
└── 底部协议勾选
```

### 2.2 数据来源
- 演示数据，无需后端接口

## 🎨 布局与结构

### 3.1 整体布局
- **布局模式**：单栏居中
- **容器宽度**：max-width 420px
- **关键尺寸**：品牌区占上半部分，操作区居下
