# 人才数字档案管理台

## 📋 业务与功能

### 1.1 核心目标
> 管理员查看和管理平台所有注册人员基本信息、证书持有情况、全景数字档案、审核事项的后台页面。对应需规 4.6 节。

### 1.2 功能清单
- **全局导航二级下拉菜单（Dropdown Submenu）**：在顶部导航栏「人才数字档案」菜单项下方，hover 触发展开基础信息管理 / 资格证书管理 / 全景数字档案 / 审核工作台 四个子模块
- **基础信息管理**：
  - 统计卡片区：总注册人数、本月新增、已认证人数
  - 筛选区：姓名/手机号/认证状态/用户类型/专业方向/注册时间
  - 数据表格：人员列表（头像、姓名、性别、手机号、认证状态、持有证书数等）
  - 行级操作：查看档案、编辑、冻结/解冻
- **资格证书管理**：
  - 统计卡片：有效演员证数、有效专业资格证数、即将到期数、已过期数
  - 筛选区：持证人姓名/证书类型/证书等级/证书状态/日期范围
  - 数据表格：证书列表（编号、持证人、类型、等级、发证日期等）
  - 行级操作：查看详情、暂停、恢复、吊销
- **全景数字档案**：
  - 人才概要信息（头像+姓名+健康度评分）
  - Tab 导航切换：基本信息/证书记录/培训记录/考核记录/荣誉记录/红黑榜/心理健康
  - 雷达图可视化档案健康度
- **审核工作台**：
  - Tab 分类：待审核/已处理/全部
  - 事项类型筛选
  - 数据表格
  - 审核处理弹窗

### 1.3 交互要点
- 全局导航栏「人才数字档案」菜单项 hover 展开二级下拉菜单（Dropdown Submenu）切换子模块
- 顶部导航菜单跳转其他管理台
- 查看档案：跳转至全景数字档案视图
- 冻结/解冻：二次确认弹窗
- 吊销证书：需填写原因 + 二次确认
- 审核处理：弹窗填写处理结论和意见

## 🎨 视觉规范

### 4.1 设计规范来源
**设计规范来源**：复用 `yanyuan-web-training-manage` 设计体系

### 4.2 自定义设计要点
**认证状态标签色**：
- 待审核(pending)：#FFF8E6 底色 + #F5A623 文字
- 已认证(verified)：#E6F7EE 底色 + #30A46C 文字
- 已驳回(rejected)：#FEECEC 底色 + #D13438 文字
- 已冻结(frozen)：#F5F0EC 底色 + #999 文字

**证书状态标签色**：
- 有效(valid)：#E6F7EE 底色 + #30A46C 文字
- 即将到期(expiring)：#FFF8E6 底色 + #F5A623 文字
- 已过期(expired)：#FEECEC 底色 + #D13438 文字
- 暂停(suspended)：#F5F0EC 底色 + #999 文字
- 吊销(revoked)：#2C2C2C 底色 + #fff 文字
