# 演艺人才平台 - 认证中心首页

## 📋 业务与功能

### 1.1 核心目标
>
> 认证中心为演员用户展示当前认证状态、个人信息完整度，并提供认证相关操作入口。

### 1.2 功能清单

- **认证状态卡片**：展示未认证/审核中/已认证/已驳回四种状态
- **信息完整度**：环形进度条+百分比
- **功能入口列表**：个人信息填报、实名认证、认证进度查看

### 1.3 交互要点

- 点击状态卡片上的操作按钮 → 跳转对应页面
- 点击功能入口 Cell → 跳转对应子页面

---

## 📊 内容规划

### 2.1 信息架构

```
认证中心首页
├── 认证状态卡片（大卡片）
│   ├── 状态标签（未认证/审核中/已认证/已驳回）
│   ├── 操作按钮
│   └── 驳回原因（已驳回时显示）
├── 信息完整度（环形进度）
└── 功能入口列表
    ├── 个人信息填报
    ├── 实名认证
    └── 认证进度查看
```

## ⚙️ Axure API 说明

### 5.1 事件列表

| 事件名称 | Payload 类型 | 触发时机 |
|---------|-------------|---------|
| `on_action_click` | `string` | 点击状态卡片操作按钮 |
| `on_entry_click` | `string` | 点击功能入口项 |
| `on_tab_change` | `string` | 切换底部Tab |

### 5.4 配置项列表

| 配置项名称 | 类型 | 默认值 | 说明 |
|----------|-----|-------|------|
| `auth_status` | `string` | `authenticated` | 认证状态 |
| `info_completeness` | `number` | `78` | 信息完整度(0-100) |
| `user_name` | `string` | `张明远` | 用户姓名 |
