# 中国演艺人才管理与服务平台 - 登录/注册中心

## 📋 业务与功能

### 1.1 核心目标
> 平台统一身份认证入口，支持手机号验证码登录与微信扫码登录两种方式，同时提供分步骤的新用户注册流程（手机号验证 → 基本信息填写 → 实名认证）及密码重置功能。页面需体现演艺行业的文化品牌气质，给用户留下专业可信赖的第一印象。

### 1.2 功能清单

- **登录页**：手机号验证码登录（Tab 1）、微信扫码登录（Tab 2）、记住登录状态
- **注册页**：三步分步注册（手机号验证 → 基本信息 → 实名认证）、步骤导航
- **密码重置页**：手机号验证 + 新密码设置
- **页面切换**：登录 ↔ 注册 ↔ 忘记密码之间的无缝切换

### 1.3 交互要点

- Tab 切换手机号登录 / 扫码登录，切换时有平滑过渡
- 获取验证码按钮点击后 60 秒倒计时
- 注册步骤条实时高亮当前步骤
- 表单实时校验，错误信息即时反馈
- 扫码登录区域展示动态二维码占位 + 状态文案
- 所有表单按钮 hover 微动效

## 🎨 布局与结构

### 3.1 整体布局

- **布局模式**：左右分栏（品牌展示区 + 表单操作区）
- **容器宽度**：全屏铺满，左侧品牌区占 55%，右侧表单区占 45%
- **关键尺寸**：
  - 表单卡片最大宽度：420px
  - 表单元素高度：44px
  - 按钮高度：48px
  - 左侧品牌区最小宽度：500px

### 3.2 响应式适配

- **桌面端（≥1200px）**：左右分栏布局
- **平板端（768-1199px）**：左侧品牌区缩窄，表单区居中
- **移动端（<768px）**：隐藏左侧品牌区，表单区全屏居中

## 🎨 视觉规范

### 4.1 设计规范来源

**设计规范来源**：主题 `/src/themes/antd-new/`

### 4.2 自定义设计要点

**自定义色彩**：
- 品牌主色：#B8292F（中国红）
- 品牌辅色：#C5963A（暖金色）
- 左侧品牌区背景渐变：linear-gradient(135deg, #B8292F 0%, #D4553A 40%, #C5963A 100%)
- 右侧表单区背景：#FBF9F7（暖白色）

**自定义尺寸**：
- 表单输入框高度：44px
- 主按钮高度：48px
- 表单卡片圆角：16px

**其他自定义规范**：
- 左侧品牌区使用文化纹样装饰（同心圆 SVG 纹理）
- 表单区域有微妙的玻璃拟态效果
- 密码强度指示器（弱/中/强三档）

### 4.3 组件状态

- **默认态（default）**：输入框白底灰边，按钮品牌色填充
- **悬停态（hover）**：输入框边框变主色，按钮加深
- **聚焦态（focus）**：输入框品牌色边框 + 浅色光晕
- **禁用态（disabled）**：灰色背景，不可点击
- **错误态（error）**：红色边框 + 错误提示文字
- **加载态（loading）**：按钮显示加载动画
