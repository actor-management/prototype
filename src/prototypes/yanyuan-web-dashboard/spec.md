# 中国演艺人才管理与服务平台 - 首页工作台

## 📋 业务与功能

### 1.1 核心目标
> 平台登录后的首页默认展示区域，以 Admin 管理员视角设计。聚合展示待办事项、平台运营关键指标和效率看板，是管理员的日常操作起点。同时包含全局顶部导航栏和页脚，建立整个 Web 端的界面框架。

### 1.2 功能清单

- **全局顶部导航栏**：左侧平台 Logo + 名称（点击返回首页），中部水平导航菜单（首页/认证中心/证书管理/职业能力评价/人才数字档案/行业机构管理/财务管理/后台管理），右侧消息图标 + 用户头像/昵称下拉菜单
- **数据概览卡片**：4 个统计卡片（本月新增注册、待审核证书、在审培训、本月收入），含同比数据
- **待办事项**：待审批列表（最多 10 条），包含证书申请审批、认证审批、年审审批
- **效率看板**：审批处理效率（环形图）+ 月度趋势（折线图）
- **快捷操作区**：管理入口按钮组（证书管理台、培训管理台、考核管理台、认证管理台、人才档案）
- **通知公告区**：最新 5 条通知公告，分类 Tab 切换
- **全局页脚**：版权声明、备案信息、联系方式

### 1.3 交互要点

- 导航菜单 hover 高亮，当前页面菜单项激活态
- 数据卡片 hover 微动效（上浮 + 阴影加深）
- 待办事项点击跳转详情
- 快捷操作按钮点击跳转目标页面
- 消息图标显示未读数角标
- 用户头像下拉菜单：个人中心/账号设置/退出登录

- **关键字段**：
  - `monthly_new_users`: 本月新增注册数

  - `monthly_income`: 本月收入

## 🎨 布局与结构

### 3.1 整体布局

- **布局模式**：单栏（顶部导航 + 主内容区 + 页脚）
- **容器宽度**：最大宽度 1440px，居中展示，两侧留白
- **关键尺寸**：
  - 顶部导航栏高度：64px，固定在顶部
  - 内容区水平间距：24px
  - 数据卡片：4 列栅格（lg=6 per card）
  - 待办事项与效率看板：16/8 栅格分栏
  - 快捷操作区：5 个入口均分

### 3.2 响应式适配

- **桌面端（≥1200px）**：标准布局，所有模块正常展示
- **平板端（768-1199px）**：数据卡片两列排列，主内容区纵向堆叠
- **移动端（<768px）**：单列堆叠，导航收起为汉堡菜单

## 🎨 视觉规范

### 4.1 设计规范来源

**设计规范来源**：主题 `/src/themes/antd-new/`

### 4.2 自定义设计要点

**自定义色彩**：
- 品牌主色：#C41230（中国红，体现演艺行业的庄重与权威）
- 品牌辅色：#D4380D（暖橙红，用于强调和次要操作）
- 导航背景：#1A1A2E（深邃蓝黑，体现专业感）
- 导航文字：rgba(255,255,255,0.85)

**自定义尺寸**：
- 顶部导航高度：64px
- 内容区最大宽度：1440px
- 卡片圆角：12px

**其他自定义规范**：
- 品牌 Logo 区使用中国红色调
- 导航栏采用深色背景以突出品牌感
- 数据卡片使用渐变色图标背景

### 4.3 组件状态

- **默认态（default）**：卡片白底无阴影，表格普通展示
- **悬停态（hover）**：卡片微上浮 + 淡阴影，按钮色彩变化
- **选中态（selected）**：导航菜单项底部高亮条
- **加载态（loading）**：骨架屏占位
